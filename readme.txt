Chạy web khi đang trong thư mục /selenium
Nhập lệnh npm run start
Web chạy trên http://localhost:10000/
Kiểm tra bằng Selenium bao gồm (đăng nhập,quản lý sản phẩm) thông qua file login.js,new-product.js
Chạy lệnh "node login.js" để kiểm tra đăng nhập.
Chạy lệnh "node new-product.js" để kiểm tra đăng sản phẩm mới.
Chạy lệnh "node edit-product.js" để kiểm tra sửa sản phẩm sản phẩm mới.
